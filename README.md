
# 🚀 Resume Analyzer using NLP and ATS Scoring

An intelligent Resume Analysis System that leverages **Natural Language Processing (NLP)** to evaluate resumes, extract key information, calculate ATS scores, identify skill gaps, and assist recruiters in candidate screening.

The system provides separate interfaces for **Students** and **Recruiters**, enabling efficient resume evaluation, candidate ranking, and hiring decision support.

---

## 🎯 Project Objective

Traditional resume screening is time-consuming and prone to human bias. This project automates the resume evaluation process by analyzing resumes against job requirements and generating meaningful insights.

The system helps:

* Students improve their resumes
* Recruiters shortlist candidates efficiently
* Organizations streamline the hiring process

---
